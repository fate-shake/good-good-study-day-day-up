# css3新增属性
### transform
transform，元素变形效果，可以将元素实现旋转、缩放、平移的效果；

平移：translate、translateX、translateY

缩放：scale、scaleX、scaleY

旋转：rotate

### transition
实现属性变化时的平滑过渡功能
transition-property：指定需要实现过渡功能的css属性

transition-duration: 平滑过渡的时间

transition-timing-function：制定过渡的函数

transition-delay：过渡开始出现的时间

可以按上面属性值连写多个属性

### animation
动画效果
